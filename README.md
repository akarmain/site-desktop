# akarmain.dev

Сайт-визитка разработчика Андрея Кармаева во второй версии: интерактивный рабочий стол в стиле `akarmainOS` с окнами контактов, проектов, стека, услуг, опыта и терминала.

## О проекте

Проект собран на Nuxt и Vue. Интерфейс имитирует персональную desktop-среду: окна можно открывать из дока, перетаскивать, разворачивать и использовать для быстрого просмотра информации о разработчике, услугах и ссылках.

Основной фокус сайта:

- показать профиль fullstack-разработчика;
- собрать контакты и социальные ссылки в одном месте;
- представить стек, услуги, опыт и примеры проектов;
- оставить ощущение живого, нестандартного портфолио вместо обычного лендинга.

## Технологии

- Nuxt
- Vue
- TypeScript
- Tailwind CSS
- shadcn-nuxt

## Запуск

Установка зависимостей:

```bash
pnpm install
```

Запуск dev-сервера:

```bash
pnpm dev
```

Сборка production-версии:

```bash
pnpm build
```
